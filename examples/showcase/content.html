Content Widgets

Cards, stats, timelines, carousels, ratings, and tree views for displaying rich content.

Cards

Versatile card containers with titles, content, footers, and collapsible support.

View Code 
Teryx.card('#card-1', {
  title: 'Getting Started',
  content: '<p>Welcome to Teryx. Set up your first widget in minutes.</p>',
  footer: '<small>Updated 2 hours ago</small>',
});

Teryx.card('#card-2', {
  title: 'System Status',
  content: '<p>All services operational. Uptime: 99.98%</p>',
  collapsible: true,
});

Teryx.card('#card-3', {
  title: 'Quick Actions',
  content: '<p>Create users, manage roles, and configure settings.</p>',
  closable: true,
});

Stat / KPI Cards

Key performance indicators with values, change indicators, icons, and sparklines.

View Code 
Teryx.stat('#stat-1', {
  label: 'Total Users', value: '12,847', icon: 'user', color: 'primary',
  change: '+14.2%', changeType: 'up',
  sparkline: [30, 40, 35, 50, 49, 60, 70, 65, 80, 75, 90, 100],
});

Teryx.stat('#stat-2', {
  label: 'Revenue', value: '$284,500', icon: 'home', color: 'success',
  change: '+8.7%', changeType: 'up', prefix: '',
});

Teryx.stat('#stat-3', {
  label: 'Active Sessions', value: '1,247', icon: 'eye', color: 'info',
  change: '+3.1%', changeType: 'up',
});

Teryx.stat('#stat-4', {
  label: 'Error Rate', value: '0.12%', icon: 'info', color: 'danger',
  change: '-2.4%', changeType: 'down',
});

Timeline

Chronological list of events with status indicators, icons, and colored markers.

View Code 
Teryx.timeline('#demo-timeline', {
  items: [
    { title: 'Application Deployed', content: 'v2.4.1 deployed to production', time: 'Today, 2:30 PM', status: 'completed', color: 'success' },
    { title: 'Code Review Approved', content: 'PR #847 approved by 2 reviewers', time: 'Today, 11:15 AM', status: 'completed', color: 'info' },
    { title: 'Build Started', content: 'CI pipeline running tests', time: 'Today, 10:45 AM', status: 'active', color: 'warning' },
    { title: 'Feature Branch Created', content: 'feature/user-dashboard', time: 'Yesterday, 4:00 PM', status: 'completed' },
    { title: 'Sprint Started', content: 'Sprint 24 kicked off', time: '3 days ago', status: 'completed' },
  ],
});

Carousel

Slide-based content carousel with navigation arrows, dot indicators, and autoplay.

View Code 
Teryx.carousel('#demo-carousel', {
  autoplay: true,
  interval: 5000,
  indicators: true,
  arrows: true,
  loop: true,
  slides: [
    { title: 'Enterprise Data Grids', description: 'Sort, filter, paginate, and export with ease.',
      content: '<div style="...">...</div>' },
    { title: 'Pure SVG Charts', description: 'Bar, line, area, pie, donut, scatter, and gauge charts.' },
    { title: 'Mobile-First Widgets', description: 'Touch-optimized components for iOS and Android.' },
  ],
});
