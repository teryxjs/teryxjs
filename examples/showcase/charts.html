Charts

Pure SVG chart rendering with no external dependencies. Supports bar, line, area, pie, donut, scatter, and gauge types.

Bar Chart

Vertical bar chart with multiple series comparing revenue and expenses.

View Code 
Teryx.chart('#demo-bar-chart', {
  type: 'bar',
  title: 'Monthly Revenue vs Expenses',
  height: 350,
  series: [
    { name: 'Revenue', data: [
      { x: 'Jan', y: 4200 }, { x: 'Feb', y: 5100 }, { x: 'Mar', y: 4800 },
      { x: 'Apr', y: 6200 }, { x: 'May', y: 5900 }, { x: 'Jun', y: 7400 },
    ]},
    { name: 'Expenses', data: [
      { x: 'Jan', y: 3200 }, { x: 'Feb', y: 3800 }, { x: 'Mar', y: 3500 },
      { x: 'Apr', y: 4100 }, { x: 'May', y: 3900 }, { x: 'Jun', y: 4600 },
    ]},
  ],
  yAxis: { title: 'Amount ($)' },
});

Pie & Donut Charts

Circular charts for proportional data. Donut charts include a hollow center for additional context.

View Code 
// Pie chart
Teryx.chart('#demo-pie-chart', {
  type: 'pie',
  title: 'Market Share',
  height: 300,
  data: [
    { x: 'Chrome', y: 65, label: 'Chrome' },
    { x: 'Safari', y: 18, label: 'Safari' },
    { x: 'Firefox', y: 8, label: 'Firefox' },
    { x: 'Edge', y: 5, label: 'Edge' },
    { x: 'Other', y: 4, label: 'Other' },
  ],
});

// Donut chart
Teryx.chart('#demo-donut-chart', {
  type: 'donut',
  title: 'Budget Allocation',
  height: 300,
  donutWidth: 0.35,
  data: [
    { x: 'Engineering', y: 45, label: 'Engineering' },
    { x: 'Marketing', y: 25, label: 'Marketing' },
    { x: 'Sales', y: 15, label: 'Sales' },
    { x: 'Operations', y: 10, label: 'Operations' },
    { x: 'HR', y: 5, label: 'HR' },
  ],
});

Scatter Chart

XY scatter plot for visualizing correlations between two numeric variables.

View Code 
Teryx.chart('#demo-scatter-chart', {
  type: 'scatter',
  title: 'Price vs Performance',
  height: 350,
  series: [
    { name: 'Product A', data: [
      { x: 10, y: 80 }, { x: 20, y: 75 }, { x: 35, y: 90 },
      { x: 45, y: 85 }, { x: 55, y: 92 }, { x: 70, y: 88 },
    ]},
    { name: 'Product B', data: [
      { x: 15, y: 60 }, { x: 25, y: 70 }, { x: 40, y: 65 },
      { x: 50, y: 78 }, { x: 60, y: 72 }, { x: 75, y: 82 },
    ]},
  ],
  xAxis: { title: 'Price ($)' },
  yAxis: { title: 'Performance Score' },
});

Gauge Chart

Semi-circular gauge for displaying a single metric against a range. Color changes based on value threshold.

View Code 
Teryx.chart('#demo-gauge1', {
  type: 'gauge',
  title: 'CPU Usage',
  height: 260,
  gaugeMin: 0,
  gaugeMax: 100,
  gaugeValue: 73,
  gaugeLabel: 'CPU %',
});

Teryx.chart('#demo-gauge2', {
  type: 'gauge',
  title: 'Disk Space',
  height: 260,
  gaugeMin: 0,
  gaugeMax: 500,
  gaugeValue: 342,
  gaugeLabel: 'GB Used',
});

Chart from Remote API

Chart data fetched from a server endpoint using the source option.

View Code 
Teryx.chart('#demo-remote-chart', {
  type: 'bar',
  title: 'Revenue vs Expenses (API Data)',
  height: 350,
  source: '/api/chart/revenue',
  yAxis: { title: 'Amount ($)' },
});
