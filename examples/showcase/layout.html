Layout Widgets

Containers and navigation structures for organizing application content.

Tabs

Tab panel with multiple style variants. Content loads when the tab is selected.

View Code 
Teryx.tabs('#demo-tabs', {
  variant: 'underline',
  items: [
    { id: 'overview', title: 'Overview', active: true,
      content: '<div class="panel-content">Overview of the application metrics and key performance indicators for the current quarter.</div>' },
    { id: 'analytics', title: 'Analytics', icon: 'eye',
      content: '<div class="panel-content">Detailed analytics showing user behavior, conversion rates, and traffic sources over time.</div>' },
    { id: 'reports', title: 'Reports', badge: '3',
      content: '<div class="panel-content">Generated reports ready for download. Three new reports are available since your last visit.</div>' },
    { id: 'settings', title: 'Settings', icon: 'settings',
      content: '<div class="panel-content">Configure dashboard preferences, notification settings, and data refresh intervals.</div>' },
  ],
});

Accordion

Collapsible panels for organizing content into sections. Supports multiple or single panel mode.

View Code 
Teryx.accordion('#demo-accordion', {
  bordered: true,
  items: [
    { id: 'general', title: 'General Settings', open: true,
      content: '<div class="panel-content">Application name, language, timezone, and default currency settings.</div>' },
    { id: 'security', title: 'Security', icon: 'eye',
      content: '<div class="panel-content">Two-factor authentication, session timeout, IP allowlisting, and audit logging.</div>' },
    { id: 'notifications', title: 'Notifications',
      content: '<div class="panel-content">Email alerts, push notifications, Slack integration, and digest frequency.</div>' },
    { id: 'billing', title: 'Billing & Invoices',
      content: '<div class="panel-content">Payment methods, billing history, invoice downloads, and subscription management.</div>' },
  ],
});

Sidebar

The sidebar navigation you see on the left of this page is a live Teryx sidebar widget. It supports collapsible sections, nested items, badges, and dark/light variants.

Main content area

Steps / Wizard

Multi-step progress indicator for guiding users through a sequential process.

View Code 
const wizard = Teryx.steps('#demo-steps', {
  current: 0,
  items: [
    { title: 'Account', description: 'Create your account', status: 'finish' },
    { title: 'Profile', description: 'Set up your profile', status: 'process' },
    { title: 'Preferences', description: 'Choose your settings' },
    { title: 'Complete', description: 'Review and finish' },
  ],
});

// Navigate with buttons
document.getElementById('steps-next').onclick = () => wizard.next();
document.getElementById('steps-prev').onclick = () => wizard.prev();

Splitter

Resizable split panel. Drag the divider to resize the two panes.

Left Panel
This is the navigation or sidebar area. Drag the divider to resize.
Right Panel
This is the main content area. It expands and contracts as you drag.
View Code 
Teryx.splitter('#demo-splitter', {
  orientation: 'horizontal',
  sizes: [30, 70],
  minSize: 100,
  gutterSize: 6,
});

Drawer

Slide-out panel from any edge. Click the buttons to open drawers from different positions.

View Code 
const leftDrawer = Teryx.drawer(document.body, {
  title: 'Left Drawer',
  position: 'left',
  content: '<div style="padding:1rem">Navigation menu or filter panel.</div>',
});
document.getElementById('btn-drawer-left').onclick = () => leftDrawer.open();
