Mobile Widgets

Touch-optimized components designed for mobile applications. Shown inside phone-sized frames.

Navigation View

iOS-style push/pop navigation with animated transitions and automatic back button. Click items to navigate deeper.

View Code 
const nav = Teryx.frontoffice.mobile.navigationView('#container', {
  items: [{
    id: 'home',
    title: 'Inbox',
    content: '<div>...list of messages...</div>',
  }],
});

// Push a detail view
nav.push({
  id: 'detail-1',
  title: 'Message Detail',
  content: '<div>...message content...</div>',
});

Bottom Sheet

Action sheet that slides up from the bottom. Supports actions list, drag-to-dismiss, and custom content.

View Code 
const actionSheet = Teryx.frontoffice.mobile.sheet({
  title: 'Share Document',
  message: 'Choose how you want to share this file',
  actions: [
    { label: 'Copy Link', icon: 'file' },
    { label: 'Email', icon: 'email' },
    { label: 'Download', icon: 'download' },
    { label: 'Delete', icon: 'trash', destructive: true },
  ],
});
actionSheet.open();

Pull-to-Refresh List

Mobile list with pull-to-refresh, infinite scroll, and swipe actions. The list fetches data from an API.

Messages
View Code 
Teryx.frontoffice.mobile.pullList('#demo-pull-list', {
  source: '/api/messages',
  itemsField: 'items',
  pullToRefresh: true,
  infiniteScroll: true,
  pageSize: 10,
  itemTemplate: '<div class="msg-item">' +
    '<div class="msg-avatar">{{initials}}</div>' +
    '<div class="msg-body">' +
    '<div class="msg-sender">{{sender}}</div>' +
    '<div class="msg-subject">{{subject}}</div>' +
    '</div><span class="msg-time">{{time}}</span></div>',
});

Bottom Tabs

Fixed bottom tab bar for app-level navigation with icons, labels, and badge indicators.

View Code 
Teryx.frontoffice.mobile.bottomTabs('#container', {
  items: [
    { id: 'home', label: 'Home', icon: 'home', active: true,
      content: '<div style="padding:1.5rem; text-align:center">Home Screen</div>' },
    { id: 'search', label: 'Search', icon: 'search',
      content: '<div style="padding:1.5rem; text-align:center">Search Screen</div>' },
    { id: 'inbox', label: 'Inbox', icon: 'email', badge: '5',
      content: '<div style="padding:1.5rem; text-align:center">Inbox Screen</div>' },
    { id: 'profile', label: 'Profile', icon: 'user',
      content: '<div style="padding:1.5rem; text-align:center">Profile Screen</div>' },
  ],
});
