Calendar

Full calendar widget with month, week, and day views plus event management.

Month View with Events

Default month view showing events placed on their respective dates. Click events for details.

View Code 
const cal = Teryx.calendar('#demo-month-cal', {
  view: 'month',
  events: [
    { id: 'e1', title: 'Team Standup', start: '2025-03-15T09:00', color: '#3b82f6' },
    { id: 'e2', title: 'Sprint Planning', start: '2025-03-16T10:00', end: '2025-03-16T12:00', color: '#22c55e' },
    { id: 'e3', title: 'Design Review', start: '2025-03-17T14:00', color: '#f59e0b' },
    { id: 'e4', title: 'Product Launch', start: '2025-03-20', allDay: true, color: '#ef4444' },
    { id: 'e5', title: 'Board Meeting', start: '2025-03-18T13:00', color: '#8b5cf6' },
    { id: 'e6', title: 'All-Hands', start: '2025-03-22T11:00', color: '#06b6d4' },
  ],
  onEventClick: (event) => {
    Teryx.toast.info(event.title + ' clicked');
  },
});

Week View

Weekly schedule showing events positioned by time of day across the week.

View Code 
Teryx.calendar('#demo-week-cal', {
  view: 'week',
  events: [...],
  onDateClick: (date) => {
    Teryx.toast.info('Clicked: ' + date);
  },
});

Day View

Single-day view with hourly time slots showing today's schedule.

View Code 
Teryx.calendar('#demo-day-cal', {
  view: 'day',
  events: [...],
  onEventClick: (event) => {
    Teryx.messageBox({
      title: event.title,
      message: 'Start: ' + event.start,
      type: 'info',
    });
  },
});
