Data Widgets

Property grids, description lists, data lists, pagination, and data export utilities.

Property Grid

Editable name-value grid for displaying and modifying object properties, organized by groups.

View Code 
Teryx.propertyGrid('#demo-propgrid', {
  editable: true,
  grouped: true,
  nameWidth: '45%',
  properties: [
    { name: 'name', value: 'ProductWidget', type: 'string', label: 'Name', group: 'General' },
    { name: 'version', value: '2.4.1', type: 'string', label: 'Version', group: 'General' },
    { name: 'enabled', value: true, type: 'boolean', label: 'Enabled', group: 'General' },
    { name: 'width', value: 800, type: 'number', label: 'Width (px)', group: 'Layout' },
    { name: 'height', value: 600, type: 'number', label: 'Height (px)', group: 'Layout' },
    { name: 'theme', value: 'light', type: 'select', label: 'Theme', group: 'Appearance',
      options: [{ label: 'Light', value: 'light' }, { label: 'Dark', value: 'dark' }] },
    { name: 'accentColor', value: '#3b82f6', type: 'color', label: 'Accent Color', group: 'Appearance' },
  ],
  onChange: (name, value) => console.log('Changed:', name, '=', value),
});

Descriptions

Read-only key-value display for entity details. Supports multi-column layouts and bordered styles.

View Code 
Teryx.descriptions('#demo-descriptions', {
  title: 'Server Information',
  columns: 2,
  bordered: true,
  items: [
    { label: 'Hostname', value: 'prod-web-01.acme.com' },
    { label: 'IP Address', value: '10.0.1.42' },
    { label: 'Operating System', value: 'Ubuntu 22.04 LTS' },
    { label: 'CPU', value: '8 vCPU (Intel Xeon)' },
    { label: 'Memory', value: '32 GB DDR4' },
    { label: 'Storage', value: '500 GB NVMe SSD' },
    { label: 'Uptime', value: '142 days, 7 hours' },
    { label: 'Last Updated', value: 'March 12, 2025' },
    { label: 'Region', value: 'us-east-1', colspan: 2 },
  ],
});

Data List

Templated data list rendering from a remote API. Supports list and grid layouts.

View Code 
Teryx.datalist('#demo-datalist', {
  source: '/api/notifications',
  itemsField: 'items',
  layout: 'list',
  itemTemplate: '<div style="padding:0.75rem; border-bottom:1px solid #f1f5f9;">' +
    '<strong>{{title}}</strong><br>' +
    '<small style="color:#94a3b8">{{time}}</small></div>',
  emptyMessage: 'No notifications',
});

Simple mode:

Teryx.pagination('#pag-2', {
  total: 500, current: 3, pageSize: 20,
  showSizeChanger: true, showTotal: true, showJumper: true,

Data Export

Export utilities for converting grid data to CSV, JSON, and HTML formats.

Click the buttons to export the sample data below:

View Code 
// Define columns for export
const exportColumns = [
  { field: 'name', label: 'Name' },
  { field: 'email', label: 'Email' },
  { field: 'role', label: 'Role' },
  { field: 'status', label: 'Status' },
];

// Export functions
const data = [
  { name: 'Alice Johnson', email: 'alice@acme.com', role: 'Admin', status: 'Active' },
  { name: 'Bob Smith', email: 'bob@acme.com', role: 'Editor', status: 'Active' },
  { name: 'Carol Williams', email: 'carol@acme.com', role: 'Viewer', status: 'Inactive' },
];

document.getElementById('btn-export-csv').onclick = () => {
  const csv = Teryx.exportCSV(data, Teryx.gridColumnsToExport(exportColumns));
  downloadFile(csv, 'export.csv', 'text/csv');
};

document.getElementById('btn-export-json').onclick = () => {
  const json = Teryx.exportJSON(data);
  downloadFile(json, 'export.json', 'application/json');
};
