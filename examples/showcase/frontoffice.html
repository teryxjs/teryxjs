Front Office Widgets

Landing page sections for marketing sites: hero areas, pricing tables, testimonials, feature grids, and footers.

Hero Section

Full-width hero with title, subtitle, CTA buttons, and gradient background.

View Code 
Teryx.frontoffice.desktop.hero('#demo-hero', {
  title: 'Build Faster with Teryx',
  subtitle: 'Enterprise-grade UI components that work out of the box. Ship products 10x faster.',
  backgroundGradient: 'linear-gradient(135deg, #1e3a5f 0%, #2d5a87 50%, #1a365d 100%)',
  minHeight: '400px',
  align: 'center',
  buttons: [
    { label: 'Get Started Free', variant: 'primary', href: '#' },
    { label: 'View Documentation', variant: 'outline', href: '#' },
  ],
  extraContent: '<p style="...">Trusted by 2,000+ companies worldwide</p>',
});
