Feedback Widgets

Modals, notifications, alerts, progress indicators, dropdowns, and context menus.

Modal / Dialog

Modal windows with customizable sizes, dragging, and button actions.

View Code 
const mdl = Teryx.modal(document.body, {
  title: 'Confirm Action',
  content: '<p>Are you sure you want to proceed?</p>',
  size: 'md',
  buttons: [
    { label: 'Cancel', variant: 'secondary', action: 'close' },
    { label: 'Confirm', variant: 'primary', handler: () => { Teryx.toast.success('Confirmed!'); mdl.close(); } },
  ],
});
mdl.open();

Toast Notifications

Temporary notification messages that appear and auto-dismiss. Four types available.

View Code 
Teryx.toast.info('This is an informational message');
Teryx.toast.success('Operation completed successfully', { title: 'Success' });
Teryx.toast.warning('Please review before continuing');
Teryx.toast.danger('Something went wrong', { duration: 8000 });

Message Box

System-style alert and confirmation dialogs with typed icons.

View Code 
Teryx.messageBox({
  title: 'Delete Item',
  message: 'This action cannot be undone. Are you sure?',
  type: 'warning',
  icon: true,
  buttons: [
    { text: 'Cancel', value: 'cancel', variant: 'secondary' },
    { text: 'Delete', value: 'delete', variant: 'danger' },
  ],
  onResult: (btn) => Teryx.toast.info('You clicked: ' + btn),
});

Alert

Inline alert banners for contextual messages. Can be dismissible.

View Code 
Teryx.alert('#container', { type: 'info', message: 'Your trial expires in 7 days.', dismissible: true, icon: true });
Teryx.alert('#container', { type: 'success', message: 'Profile updated successfully.', icon: true });
Teryx.alert('#container', { type: 'warning', message: 'Low disk space on server.', title: 'Warning', icon: true });
Teryx.alert('#container', { type: 'danger', message: 'Payment failed. Please update your card.', icon: true, dismissible: true });

Progress Bar

Progress indicators with various sizes, colors, and animation options.

View Code 
Teryx.progress('#p1', { value: 25, showValue: true, label: 'Upload Progress', size: 'sm' });
Teryx.progress('#p2', { value: 60, showValue: true, color: 'success', size: 'md' });
Teryx.progress('#p3', { value: 85, showValue: true, color: 'warning', striped: true, animated: true });
Teryx.progress('#p4', { value: 45, showValue: true, color: 'danger', size: 'lg' });

Dropdown Menu

Click-triggered dropdown menus with icons, nested items, and dividers.

View Code 
Teryx.dropdown({
  trigger: '#btn-dropdown',
  items: [
    { label: 'Edit', icon: 'edit' },
    { label: 'Duplicate', icon: 'file' },
    { divider: true },
    { label: 'Archive', icon: 'folder' },
    { label: 'Delete', icon: 'trash', handler: () => Teryx.toast.danger('Deleted!') },
  ],
});

Context Menu

Right-click the area below to see a context menu with custom actions.

Right-click here to open the context menu
View Code 
Teryx.contextMenu({
  trigger: '#context-demo-area',
  event: 'contextmenu',
  items: [
    { label: 'Cut', icon: 'edit', shortcut: 'Ctrl+X' },
    { label: 'Copy', icon: 'file', shortcut: 'Ctrl+C' },
    { label: 'Paste', icon: 'folder', shortcut: 'Ctrl+V' },
    { divider: true },
    { label: 'Select All', shortcut: 'Ctrl+A' },
    { divider: true },
    { label: 'Properties', icon: 'settings' },
  ],
});
