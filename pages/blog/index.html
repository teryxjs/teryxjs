March 17, 2026 Release

Teryx v0.1.0 — Initial Release

The first public release of Teryx — an enterprise-grade widget framework built on xhtmlx with TypeScript, zero dependencies, and dual declarative/imperative APIs.

Core Framework

  • Widget registry with auto-discovery via data-tx-widget attributes
  • Event bus for cross-widget communication
  • Theme system with light, dark, and high-contrast modes
  • Internationalization support with locale management
  • Client-side router for single-page applications

30+ Widgets

  • Data Display: Grid (sorting, filtering, pagination, editing), Tree, Data List, Property Grid, Descriptions
  • Navigation: Tabs, Accordion, Steps/Wizard, Sidebar, Navbar, Pagination
  • Feedback: Toast, Alert, Modal/Dialog, Message Box, Progress, Skeleton
  • Data Entry: Form (20+ field types), Date Picker, Color Picker, Slider, Rating, Tag Input, Segmented Control
  • Layout: Card, Drawer, Splitter, Carousel, Lightbox
  • Utilities: Tooltip, Popover, Context Menu, Dropdown, Drag & Drop, Export (CSV/Excel/JSON/HTML)

Build Output

  • ESM, CJS, and IIFE formats via tsup
  • TypeScript declarations for all widgets and options
  • 265KB ESM bundle (all widgets included)
